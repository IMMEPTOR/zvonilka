# zvonilka

Приложение для Android, которое блокирует входящие звонки с номеров, отсутствующих в контактах.

## Описание

На главном экране приложения расположена одна кнопка-переключатель. Когда защита включена, каждый входящий вызов проверяется: если номер звонящего сохранён в контактах устройства, звонок проходит как обычно; если номера в контактах нет — вызов отклоняется автоматически, а пользователь получает уведомление с указанием заблокированного номера.

Проверка выполняется локально на устройстве. Приложение не использует внешние базы спам-номеров, не отправляет данные на серверы и не требует интернет-соединения.

## Установка

Готовый APK-файл доступен в разделе [Releases](../../releases/latest).

1. Скачать `zvonilka-1.0.apk` на устройство.
2. Открыть файл и разрешить установку из неизвестных источников, если потребуется.
3. Запустить приложение и нажать кнопку включения.
4. При первом запуске подтвердить запрашиваемые разрешения:
   - доступ к контактам;
   - отправка уведомлений (на Android 13 и выше);
   - назначение zvonilka приложением по умолчанию для фильтрации вызовов.

## Требования

Android 10 (API 29) или новее.

## Сборка

```bash
git clone https://github.com/IMMEPTOR/zvonilka.git
cd zvonilka
echo "sdk.dir=/path/to/Android/sdk" > local.properties
./gradlew :app:assembleDebug
```

Итоговый APK будет размещён в `app/build/outputs/apk/debug/`.
